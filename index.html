Quick Start

几行代码，接入所有 AI

兼容 OpenAI SDK，无需修改现有代码

import openai client = openai.OpenAI( base_url="http://localhost:8080/v1", api_key="nex-your-key" ) response = client.chat.completions.create( model="gpt-4o", messages=[{"role": "user", "content": "Hello!"}] ) # 原生 OpenAI SDK，直接将 base_url 指向 AISIX AI Gateway 即可
import OpenAI from 'openai'; const client = new OpenAI({ baseURL: 'http://localhost:8080/v1', apiKey: 'nex-your-key', }); const response = await client.chat.completions.create({ model: 'gpt-4o', messages: [{ role: 'user', content: 'Hello!' }], }); // 零改动迁移——只需更换 baseURL
curl http://localhost:8080/v1/chat/completions \ -H "Authorization: Bearer nex-your-key" \ -H "Content-Type: application/json" \ -d '{ "model": "gpt-4o", "messages": [{"role": "user", "content": "Hello!"}] }'

兼容 OpenAI SDK — 无需修改任何现有代码，直接替换 base_url 即可
